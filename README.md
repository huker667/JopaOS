# JopaOS
JopaOS this is a Lua-System OpenComputers mod in Minecraft.

# Install
1. Install OpenOS on your computer.
2. Just write command in terminal "pastebin run B0ceSExu".
3. Done.

!!!WARNING!!! COMPUTER NEED INTERNET-CARD!
